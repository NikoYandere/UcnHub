Description

The Yanix Launcher runs Yandere Simulator on Linux using WINE. Requires WINE 8.0+. To fix missing letters: winetricks allfonts.

Install Commands

ucn add-repo https://nikoyandere.github.io/UcnHub && ucn install --from-repos org.nikoyandere.yanix-launcher
