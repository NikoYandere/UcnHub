Description

The Yanix Launcher is an independent initiative created to run Yandere Simulator on Linux operating systems. It is neither supported nor developed by YandereDev — The Yanix Launcher runs the game using WINE, and requires WINE 8.0 or newer for proper functionality. While most major display bugs like fullscreen issues have been fixed in the Unity 6 Build, some letter display problems persist. To fix the missing letters, try running winetricks allfonts in your terminal. This installs common Windows fonts which may resolve the issue. Keep in mind that full resolution of these problems would still require YandereDev's investigation, as it's beyond the scope of the launcher.

ucn add-repo https://nikoyandere.github.io/ucn-repo ucn install --from-repos org.nikoyandere.yanix-launcher
